Advice

Key learnings from using Copilot

  • Don't rely on Copilot
  • Copilot has a limited prompt length
  • Long prompts get it to be more forgetful
  • It forgets about context easily (creates new tables when they already exist)
  • Even the English language is in preview

Key learnings from using PowerApps

  • Create a clean environment
  • Multiple users might be quicker
  • Have a clear idea of what you are trying to achieve
  • Use Lookup tables not Multiple Choice
  • Existance of data in a PowerApp prevents you from deleting things

Key learnings from making a chatbot

  • Reading the website is easy
  • Connecting to data sources is okay
  • Personal OneDrive isn't good for AI
  • Organise your internal IT data
  • Integration with dynamics is less reliable
